# 이벤트 버스 (Event Bus)

> **Status**: Done
> **Author**: Juwon + agents
> **Last Updated**: 2026-04-26
> **Implements Pillar**: 인프라 — 직접적인 필라 구현이 아닌 시스템 간 결합도를 낮추는 통신 레이어

## Overview

이벤트 버스는 결합도가 높은 직접 메서드 호출 없이 시스템 간 신호를 전달하는 Core 레이어 Autoload 싱글톤이다. 발행자(Publisher)는 이벤트 수신자를 알 필요 없이 신호를 발행하고, 수신자(Subscriber)는 발행자를 참조하지 않고 신호를 구독한다.

MVP에서 이벤트 버스가 담당하는 신호는 동료 합류 요청 하나다. 추후 시스템 간 연결이 필요한 이벤트가 생기면 여기에 추가한다.

**패턴:**
```
동료-영입-퀘스트 (발행자)
  EventBus.emit_signal("companion_join_requested", companion_id)
                    ↓
  동료-합류-이벤트 (수신자)
  EventBus.companion_join_requested.connect(_on_companion_join_requested)
```

## Player Fantasy

플레이어에게 직접 보이지 않는 인프라 시스템. 하지만 이것이 없으면 동료 합류 이벤트의 두 시스템이 서로를 직접 참조해야 하고, 한 쪽이 없으면 다른 쪽도 로드되지 않는 강한 결합이 생긴다.

## Detailed Rules

**R-1: EventBus Autoload**
`EventBus`로 등록. GDScript `Node`를 상속하는 스크립트. 모든 신호를 이 노드에 선언.

**R-2: MVP 신호 목록**
```gdscript
signal companion_join_requested(companion_id: StringName)
```

| 신호 | 발행자 | 수신자 | 의미 |
|------|--------|--------|------|
| `companion_join_requested` | 동료-영입-퀘스트 (완료 대화 on_complete) | 동료-합류-이벤트 | 동료 합류 처리 시작 요청 |

**R-3: 신호 추가 규칙**
새 시스템 간 통신이 필요할 때 이 파일에 신호를 추가하고 Dependencies 테이블을 업데이트한다. 단순 부모-자식 관계의 신호는 EventBus를 거치지 않고 직접 연결한다 (EventBus는 서로를 모르는 시스템 간 통신에만 사용).

## Formulas

## Edge Cases

**EC-1: 수신자가 없는 신호 발행**
Godot 신호는 수신자 없이 발행 가능. 무시됨. 오류 아님.

**EC-2: 씬 전환 중 신호 발행**
Autoload이므로 씬 전환 영향 없음. 수신자가 씬에 속한 노드라면 씬 언로드 시 자동 연결 해제됨 — Godot의 기본 동작.

**EC-3: 동일 수신자 중복 연결**
`connect()` 시 `CONNECT_ONE_SHOT` 또는 중복 확인 없이 연결하면 수신자가 두 번 호출될 수 있음. 수신자 측에서 연결 시 `is_connected()` 확인 후 연결 권장.

## Dependencies

**이 시스템이 의존하는 것 (Upstream)**
- 없음. 순수 통신 채널. 다른 시스템의 데이터에 의존하지 않음.

**이 시스템에 의존하는 것 (Downstream)**

| 시스템 | 의존 내용 |
|--------|-----------|
| **동료-영입-퀘스트** | `companion_join_requested` 신호 발행 |
| **동료-합류-이벤트** | `companion_join_requested` 신호 구독 |

**공개 인터페이스 요약**
```gdscript
# MVP 신호
signal companion_join_requested(companion_id: StringName)
```

## Acceptance Criteria

| # | 조건 | 검증 방법 |
|---|------|-----------|
| AC-1 | `EventBus.emit_signal("companion_join_requested", id)` 발행 시 구독자가 신호를 수신한다 | 단위 테스트 |

# 군웅의 계절 (Season of Warlords)

Godot 4.6.2 + GDScript로 만드는 인디 전략 RPG 프로토타입입니다.

## 프로젝트 개요
- 이 게임은 **삼국지 세계관 게임이 아닙니다.**
- **삼국지3**는 세력 운영 구조를 참고한 레퍼런스입니다.
- 핵심 전투 감각은 **퍼스트퀸4의 실시간 부대 전투**에서 더 큰 비중으로 참고합니다.
- 세계관은 **중세 판타지**입니다.
- 현재 단계는 거대한 시스템 대신, 한 바퀴 도는 **MVP 핵심 루프**를 안정화하는 것입니다.

## 현재 MVP 목표
월드맵 선택 → 전투 진입 → 승패 판정 → 지역 점령

## 실행 방법
1. Godot 4.x 에디터에서 이 폴더를 엽니다.
2. 프로젝트 메인 씬은 `WorldMap.tscn`으로 설정되어 있습니다.
3. 실행(F5)하면 월드맵에서 시작합니다.
4. 플레이어 소유 지역(초록) 클릭 후, 인접한 적 지역(빨강/파랑)을 클릭하면 전투에 진입합니다.

## 현재 구현된 기능
- `GameState` 오토로드 싱글톤으로 전역 게임 상태 관리
- 8개 중세 판타지 지역 데이터(이름/소유 세력/인접 관계/좌표)
- 월드맵에서 인접성 기반 공격 가능 체크
- 전투 씬 10 vs 10 자동전투
- 가장 가까운 적 추적, 사거리 내 자동공격, 쿨타임 처리
- 승리 시 지역 소유권 변경 후 월드맵 반영

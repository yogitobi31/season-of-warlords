# 군웅의 계절 (Season of Warlords)

Godot 4.6 + GDScript로 만드는 중세 판타지 인디 전략 RPG 프로토타입입니다.

## 프로젝트 개요
- 이 게임은 삼국지 세계관 게임이 아닙니다.
- 삼국지3는 **세력 운영의 재미**를 참고한 레퍼런스입니다.
- 실제 컨셉 비중은 퍼스트퀸4의 **실시간 부대 전투 감각**에 더 가깝습니다.
- 세계관은 왕국, 공국, 변경백령, 성채, 요새, 항구, 산맥, 고대 유적이 등장하는 **중세 판타지**입니다.
- 현재 단계에서는 거대한 시스템 대신, 한 바퀴 도는 **MVP 핵심 루프**에 집중합니다.

## 현재 MVP 목표
월드맵 지역 선택 → 인접 지역 이동/공격 → 전투(10 vs 10 자동전투) → 승패 판정 →
승리 시 지역 점령 → 월드맵 복귀

## 실행 방법
1. Godot 4.6 에디터에서 이 폴더를 엽니다.
2. 프로젝트 메인 씬은 `WorldMap.tscn`으로 설정되어 있습니다.
3. 실행(F5)하면 월드맵에서 시작합니다.
4. 플레이어 소유 지역(청람 왕국/파랑) 클릭 후, 인접한 적 지역을 클릭하면 전투에 진입합니다.

## 현재 임시 세력
- 청람 왕국: 플레이어 세력
- 진홍 공국: 적 세력 1
- 녹림 변경백령: 적 세력 2

## 현재 임시 지역
- 청람 성채
- 북부 감시요새
- 서리숲 관문
- 서부 협곡로
- 중앙 평원
- 남해 항구
- 고대 유적지
- 녹림 산맥

## 현재 구현된 기능
- `GameState` 오토로드 싱글톤으로 전역 게임 상태 관리
- 8개 중세 판타지 지역 데이터(이름/소유 세력/인접 관계/좌표)
- 월드맵에서 인접성 기반 공격 가능 체크
- 전투 씬 10 vs 10 자동전투
- 가장 가까운 적 추적, 사거리 내 자동공격, 쿨타임 처리
- 승리 시 지역 소유권 변경 후 월드맵 반영

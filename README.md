# 군웅의 계절 (Season of Warlords)

Godot **4.6** + GDScript로 만드는 인디 전략 RPG 프로토타입입니다.

## 프로젝트 개요
- 목표: **삼국지식 세력 운영**과 **실시간 자동 부대전투**를 결합한 게임의 핵심 재미를 검증합니다.
- 현재 단계: 거대한 시스템 대신, 한 바퀴 도는 **MVP 핵심 루프**에 집중합니다.

## 현재 MVP 상태
월드맵 지역 선택 → 인접 지역 공격 → 전투(10 vs 10 자동전투) → 승패 판정 →
승리 시 지역 점령 → 월드맵 복귀

- 현재 `WorldMap`에서 지역이 화면에 표시됩니다.
- 지역 클릭으로 출발 지역 선택/인접성 검사/전투 씬 전환이 가능합니다.

## 실행 방법 (Godot 4.6 기준)
1. Godot **4.6** 에디터에서 이 폴더를 엽니다.
2. 프로젝트 메인 씬은 `res://scenes/WorldMap.tscn` 입니다.
3. `GameState`는 AutoLoad(`res://scripts/GameState.gd`)로 등록되어 있어야 합니다.
4. 실행(F5)하면 월드맵에서 시작합니다.
5. 플레이어 소유 지역(파란색) 클릭 후, 인접한 적 지역(빨강/초록/회색)을 클릭하면 전투로 이동합니다.

## 현재 구현된 기능
- `GameState` 오토로드 싱글톤으로 전역 게임 상태 관리
- 6개 지역 데이터(id/name/owner_faction/position/neighbors)
- 월드맵에서 인접성 기반 공격 가능 체크 및 상태 메시지 UI
- 전투 씬 10 vs 10 자동전투
- 승리 시 지역 소유권 변경 후 월드맵 반영

## 제한 사항
- 장수 시스템, 내정, 외교, 병과 상성, 아이템, 세력 AI는 아직 구현하지 않습니다.
- 현재 목표는 월드맵 표시/선택/전투 진입 루프의 안정화입니다.

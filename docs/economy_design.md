# Season of Warlords 경제 설계 (1차)

## 1) 현재 핵심 자원 (4종)
- **금화**: 병사 고용, 훈련, 장비 관련 범용 비용.
- **보급**: 출정 유지, 회복, 병참 운영에 필요한 유지 자원.
- **자재**: 성채 시설 강화, 제작/방어 인프라 기반 자원.
- **명성**: 청람 성채의 신뢰/평판을 나타내는 진행 지표.

## 2) Source / Sink / Gate 구조
- **Source(획득처)**: 지역 점령 보상, 이벤트 보상.
- **Sink(소모처)**: 초기에는 주로 금화/자재 기반 성채 업그레이드.
- **Gate(진행 관문)**: 명성은 초기에는 소비하지 않고, 추후 소문/동료/고급 병종 해금 조건으로 사용.

## 3) 명성을 초반에 소비하지 않는 이유
- 명성은 일반 화폐가 아니라 **신뢰와 정당성** 축적치다.
- 초반에 소비 자원으로 만들면 의미가 희석되므로, 우선은 누적형 진행 값으로 유지한다.
- 플레이어가 “무엇을 샀는가”보다 “어떤 선택과 전투로 신뢰를 얻었는가”를 명확히 보여준다.

## 4) 초반 지역 보상 철학
- 각 지역은 하나의 경제 역할(보급 확보, 자재 확보, 명성 확보, 병종 해금 등)을 갖는다.
- 보상을 명시적으로 노출해 준비 의사결정을 돕는다.
- 북부 감시요새 전까지는 분산 파밍이 아닌 **목적형 준비 루프**를 유도한다.

## 5) 확장 계획 (후속 단계)
- 장비 제작 시스템과 자재 소비처 확대.
- 대장간 시설 추가 및 단계별 강화.
- 클래스 승급(고급 병종)과 명성 연계 관문.
- 성직자(클레릭) 해금을 명성/이벤트 조건으로 연결.
- 보급을 향후 출정 비용(지속 원정 페널티 완화)으로 본격 적용.
